### Использованные данные:
* **Классификация:** Бинарная классификация бобов - реальная задача в сельком хозяйстве: https://www.kaggle.com/datasets/nimapourmoradi/dry-bean-dataset-classification
* **Регрессия:** Предсказания цены футболиста - реальная задача в футбольном бизнесе: https://www.kaggle.com/datasets/yorkyong/football-player-statistics

В результате лучше всего себя показали **логистическая регрессия** и **решающие деревья** (в случае регрессии).
Следует отметить, что результы практически всех моделей похожи. Относительная разница между основными метриками составляет не более 1% для классификации.
Для регресси же не более 10% для метрики R^2.
